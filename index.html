How To Use Ticket Logger

1. Set the Logging Channel: To log tickets into a specific channel, type: !loggerchannel IDCHANNEL Replace IDCHANNEL with the channel ID where you want tickets to appear.

2. Set the Role that Triggers the Bot: Only users with a specific role can trigger ticket logging. Type: !loggerrole @pingtherole Replace @pingtherole with the role you want to grant logging permissions.

3. Start Managing Tickets: Once configured, the bot will automatically log tickets and track staff activity in your dashboard.
